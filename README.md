## Flask MySQL JSON App
Sample json files can be found in folder Assignment_Details

#### STEP 1:
```
pip3 install -r requirements.txt
```
#### STEP 2:

Set Configurations of MySQL DB in db.yaml file

#### STEP 3:

RUN:
```
python3 app.py
```
#### STEP 4:

Navigate to http://localhost:8008/
